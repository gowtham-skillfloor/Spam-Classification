# Spam Classification Project

The objective of this project is to classify spam from non-spam emails. The data was sourced
from UCI Machine Learning repository.

Multiple machine learning models were used to improve the performance of the classification.

**Logistic Regression** model yield the highest accuracy.

Future improvements:
---
The model could be deployed in cloud environment.
More data could be collected to improve the performance of the model.

The classification accuracy was **92%**
